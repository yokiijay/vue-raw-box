# vue-raw-box
 A vue component from Vue Raw UI library.

 [See document](http://yokiijay.github.io/vue-raw-ui)

> Raw UI 是一套原始UI组件库

## 安装 Install
使用yarn安装：`yarn add vue-raw-box`

使用npm安装：`npm install vue-raw-box --save`

## 引入
```js
import Box from 'vue-raw-box'

export default {
  ...
  components: {
    Box
  },
}
```
